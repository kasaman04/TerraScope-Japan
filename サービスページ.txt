// ClaudeCode指示書：サービスページ（日本に住むことを検討する外国人向け）

/**
 * 🎯 目的
 * 日本に住むことを検討している外国人ユーザーが、
 * サービスページを通じて"この土地なら安心できる"と感じる情報を提供し、
 * 最終的にLight〜Fullプランの申し込みへと導く構成にする。
 */

/**
 * 📄 ページ構成（/services）
 * - HeroSection：安心感・信頼感を与えるメッセージとCTA
 * - PlanOverview：3プランの特徴と関係性をわかりやすく比較
 * - InvestigationItems：調査項目をカテゴリ別に紹介（FAQ形式）
 * - CaseStudyTeaser：調査事例の一部を見せて安心感を高める
 * - Flow：調査完了までの流れ（7営業日以内で完結を強調）
 * - FAQ：プラン選び・調査範囲・データ信頼性に関する不安を払拭
 * - FinalCTA：リード獲得のためのボタンと特典案内
 */

/**
 * ✅ 重点観点
 * - "住むこと"を前提とした不安を解消する情報設計（例：学校・病院・地震）
 * - 英語話者・中国語話者など非日本語ユーザーにも伝わる文体（i18n前提）
 * - "現地調査"の価値を明示し、Lightプラン記事との違いを強調
 * - フル調査まで依頼せずとも価値を感じる構成
 * - 価格に対する納得感（調査項目×サンプルレポート×安心材料）
 */

/**
 * 🧭 各セクション構成
 */

// HeroSection
- タイトル例："Is this area safe to live in? Let us investigate for you."
- サブタイトル：日本に住む前に知っておきたい地域情報を、7営業日で安心レポートに。
- CTA：無料相談 / サンプルを見る

// PlanOverview
- 表形式：Light / Standard / Full
- 横並び比較で：調査項目数、現地訪問有無、写真・映像付き、レポート構成など
- 各プランにCTA："このプランを相談"

// InvestigationItems
- カテゴリ別FAQ風見出し（6カテゴリ）：
  - 防災（地震・水害・津波）
  - 生活インフラ（病院・スーパー・学校など）
  - 土地価格と利回り
  - 再開発・都市整備の予定
  - 法制度・外国人規制
  - 現地調査：写真・動画・周囲の雰囲気

// CaseStudyTeaser
- 実際の調査レポート一部を画像またはスライドで見せる
- "ここまで細かく見てくれるなら安心"と思わせる内容に
- CTA：他の事例を見る

// Flow
- Step形式（アイコン付き）：
  1. 無料相談 →
  2. エリア選定 →
  3. 調査開始 →
  4. レポート納品
- "通常7営業日以内"でスピード感を演出

// FAQ
- 質問例：
  - 日本語が話せなくても問題ない？
  - 調査はどのくらい正確？
  - どんな土地でも対応してくれるの？
  - レポートの形式は？

// FinalCTA
- "今なら無料相談受付中！" "初回特典あり"など訴求
- CTAボタン（3プランごとに）＋ サンプルDLへの誘導

/**
 * 🎁 追加コンポーネント（ClaudeCode開発上の指示）
 * - サンプルPDFを埋め込みで表示 or サムネイル＋モーダル
 * - FAQはアコーディオン形式
 * - CTAはSanityまたはAirtableと連携できるよう送信機能付き
 */
